## 技术架构

### 前端技术栈
- Vue 3 + Vite
- Pinia（状态管理）
- Vue Router（路由）
- Element Plus（UI组件）
- Axios（HTTP客户端）
- ECharts（图表）

### 后端技术栈
- Spring Boot
- Spring Data JPA
- MySQL
- RESTful API

### AI技术栈
- 多模型支持（Ollama/OpenAI兼容API）
- 动态数据获取
- 权限边界控制
- 聊天记录持久化

## 数据库说明

- 当前后端配置为 `spring.jpa.hibernate.ddl-auto=none`
- 使用最新的 `hrms-db.sql` 数据库结构
- 审批规则基于身份标签（identity_tag_code）进行匹配
- 支持角色和身份标签双重权限管理体系

## 快速开始

### 1. 环境准备

```bash
# 安装依赖
cd HRMS-web
npm install

# 配置数据库
# 导入 hrms-db.sql 到 MySQL

# 配置AI模型（可选）
export AI_CHAT_PROVIDER_OPENAI_API_KEY=your-api-key
```

### 2. 启动服务

```bash
# 启动后端
cd HRMS-backend
./mvnw spring-boot:run

# 启动前端
cd HRMS-web
npm run dev
```

### 3. 访问系统

- 前端地址：http://localhost:5173
- 后端地址：http://localhost:8080
- 默认账号：admin / 123456
